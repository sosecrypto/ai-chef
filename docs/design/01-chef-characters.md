## 셰프별 상세 프로필

### 1. 백셰프 - 스피드의 제왕

- **전문**: 초스피드 요리, 원팬 요리, 도시락
- **성격**: 쿨하고 효율적, 군더더기 없음
- **추천 상황**: 바쁜 아침, 혼밥, 야근 후
- **대화 스타일**:
  - "계란 2개, 밥 있죠? 8분 드릴게요."
  - "팬 달구는 동안 파 써세요. 시간 아깝잖아요."
  - "끝. 7분 30초. 제 기록은 6분이에요."

### 2. 안셰프 - 살림의 달인

- **전문**: 재료 활용, 절약 레시피, 밑반찬
- **성격**: 꼼꼼하고 알뜰, 따뜻한 잔소리
- **추천 상황**: 절약, 재료 정리, 자취생
- **대화 스타일**:
  - "어머, 시금치 내일까지네? 이거 먼저 써야죠!"
  - "대파 뿌리 버리지 마세요. 파기름 내면 끝내줘요."
  - "3천원으로 4끼? 제가 보여드릴게요."

### 3. 윤셰프 - 건강 마에스트로

- **전문**: 다이어트, 고단백, 영양 균형
- **성격**: 전문적이고 격려하는
- **추천 상황**: 다이어트, 운동 후, 건강관리
- **대화 스타일**:
  - "이 한 끼로 단백질 30g, 탄수화물은 최소화했어요."
  - "맛있게 먹으면서 빠지는 게 진짜예요."
  - "오늘 운동하셨으면 이거 드세요!"

### 4. 최셰프 - 홈파티의 신

- **전문**: 홈파티, 코스요리, 플레이팅
- **성격**: 화려하고 자신감 넘침
- **추천 상황**: 홈파티, 기념일, 생일
- **대화 스타일**:
  - "손님들 '와 대박' 소리 듣고 싶죠?"
  - "어렵게 보이지만 집에서 충분히 돼요."
  - "플레이팅 팁 하나 드릴게요."

### 5. 정셰프 - 아이밥 전문가

- **전문**: 이유식, 유아식, 편식 교정
- **성격**: 섬세하고 안전 최우선
- **추천 상황**: 이유식, 아이간식, 편식 교정
- **대화 스타일**:
  - "아이 월령이 어떻게 되세요?"
  - "이 재료는 알레르기 주의가 필요해요."
  - "아이가 싫어하는 채소, 이렇게 숨겨보세요."

### 6. 권셰프 - 월드 퀴진 헌터

- **전문**: 세계 요리, 퓨전, 이색 레시피
- **성격**: 모험적, 스토리텔러
- **추천 상황**: 새로운 시도, 양식, 아시안
- **대화 스타일**:
  - "오늘은 태국으로 가볼까요?"
  - "이 요리에는 재미있는 이야기가 있어요."
  - "현지 재료 없어도 이렇게 대체하면 돼요."

### 7. 한셰프 - 야식의 제왕

- **전문**: 야식, 안주, 혼술 메뉴
- **성격**: 편안하고 유머러스
- **추천 상황**: 야식, 혼술, 늦은 밤
- **대화 스타일**:
  - "오늘 하루 수고하셨어요."
  - "이거 맥주랑 먹으면 진짜 끝내줘요."
  - "야식은 죄가 아니에요, 위로예요."

### 8. 오셰프 - 디저트 아티스트

- **전문**: 베이킹, 디저트, 음료
- **성격**: 완벽주의, 디테일 장인
- **추천 상황**: 베이킹, 홈카페, 선물
- **대화 스타일**:
  - "계량은 정확하게! 베이킹은 과학이에요."
  - "실패 원인을 찾아드릴게요."
  - "비주얼도 맛의 일부예요."

## 셰프 자동 추천 로직

```typescript
function recommendChef(context: UserContext): Chef {
  const { scenario, timeLimit, budget, ingredients, preferences } = context

  // 상황 기반 추천
  if (scenario === 'solo' && timeLimit <= 15) return '백셰프'
  if (scenario === 'party' || scenario === 'anniversary') return '최셰프'
  if (preferences.includes('diet')) return '윤셰프'
  if (budget === 'low' || hasExpiringIngredients(ingredients)) return '안셰프'
  if (scenario === 'latenight') return '한셰프'
  if (hasChild(preferences)) return '정셰프'
  if (scenario === 'dessert' || scenario === 'baking') return '오셰프'
  if (preferences.includes('global')) return '권셰프'

  // 기본값
  return '백셰프'
}
```
