# 온보딩 플로우 설계

> 첫 방문 사용자를 위한 온보딩 경험

## 전체 플로우 (5단계, 약 2분)

```
[인트로 비디오] → [환영] → [환경설정] → [셰프선택] → [냉장고] → [완료]
     10초         15초      45초        20초       30초     10초
```

---

## Step 1: 인트로 비디오 (10초)

Remotion으로 제작된 인트로 비디오
- 로고 리빌 (2초)
- 셰프 쇼케이스 (3초)
- 기능 하이라이트 (3초)
- CTA (2초)

```typescript
// user_profiles
{
  skill_level: 'novice',
  household_size: 1,
  has_children: false,
  time_preference: '10min',
  budget_preference: 'medium',
  scenarios: ['solo', 'lunchbox']
}

// cooking_tools
[
  { tool_key: 'frying_pan', is_available: true },
  { tool_key: 'pot', is_available: true },
  { tool_key: 'microwave', is_available: true },
  { tool_key: 'stove', is_available: true },
  { tool_key: 'air_fryer', is_available: false },
  { tool_key: 'oven', is_available: false }
]

// chef_settings
{
  primary_chef_id: 'baek',
  favorite_chef_ids: []
}

// ingredients (첫 냉장고 등록 시)
[
  { name: '계란', quantity: 6, unit: '개', category: 'egg' },
  { name: '양배추', quantity: 1, unit: '통', category: 'vegetable' },
  ...
]
```
